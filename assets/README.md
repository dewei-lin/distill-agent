# Assets

## `sample_outputs/`

Real output artifacts from a Distill Agent run on `examples/adult_sample.csv`
(target column `income`). Browse them here to see what the agent produces
without running anything:

| File | Artifact |
|---|---|
| `report.pdf` / `report.md` | the Cleaning Audit Report |
| `flowchart.png` / `flowchart.svg` | the CONSORT-style cleaning flowchart |
| `decisions.json` | the machine-readable decision log (38 decisions) |
| `clean_script.py` | the standalone reproducible cleaning script |
| `clean_head.csv` | first 20 rows of the cleaned dataset |

These were generated by:

```bash
python -m distill.cli clean examples/adult_sample.csv --target income
```

## `ui_preview.html`

A static snapshot of the web demo's three-panel interface (pipeline
sidebar, decision-card chat, variable inspector), populated with sample
data. Open it in a browser to see the UI without starting the server.
The live app is in `web/` — see `web/README.md`.
